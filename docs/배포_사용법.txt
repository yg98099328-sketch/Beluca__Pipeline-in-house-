========================================
  setup_pro 사용 설명서 (쉬운 버전)
========================================

이 프로그램은 "프리셋"을 저장해 두었다가
Nuke에서 한 번에 적용하는 도구예요.


■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■
  STEP 0. 프로그램 실행하기
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

이 폴더(release 배포 폴더 또는 zip 해제한 폴더)에 들어가서:

1. setup_pro_manager.exe  (또는  실행.bat)  더블클릭
   → 프로그램이 켜져요.

※ 배포받은 경우: zip 압축 해제 후, 해제된 폴더에 들어가서 실행하세요.


■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■
  STEP 1. Nuke에 메뉴 설치 (처음 한 번만!)
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

Nuke를 쓰려면 먼저 "연결"을 해줘야 해요.

1. 이 폴더 안에서  install_to_nuke.bat  찾기
2. 더블클릭해서 실행
3. Nuke를 완전히 끄고, 다시 켜기
4. Nuke 맨 위 메뉴에  setup_pro  가 보이면 성공!


■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■
  STEP 2. 프리셋 저장하기
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

setup_pro_manager.exe 를 실행했을 때:

1. "프리셋 저장 폴더" 지정
   - [변경] 버튼 눌러서 폴더 고르기
   - 팀에서 쓰는 공유 폴더를 쓰면 좋아요

2. 아래 칸들 채우기:
   - 프로젝트 타입: OTT 시리즈 / 영화 / 광고 / 기타 중 선택
   - 프로젝트 코드: 예) DRM_A01  (영문 대문자, 숫자만)
   - FPS: 23.976 / 24 / 25 등
   - OCIO Config: .ocio 파일 경로 (찾아보기로 선택)
   - 프로젝트 포맷: HD, UHD, 4K 등 또는 가로x세로 직접 입력

3. Write 세팅 (체크박스 켜기)
   - 납품 포맷, 컬러스페이스, 데이터타입 등 선택

4. [프리셋 저장] 버튼 누르기
   → "저장된 프리셋" 목록에 추가돼요.


■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■
  STEP 3. Nuke에서 프리셋 쓰기
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

1. Nuke 실행
2. 맨 위 메뉴에서  setup_pro  클릭
3.  Open setup_pro panel  클릭
4. 저장해 둔 프리셋 선택
5. 끝! Write 노드가 자동으로 만들어지고 세팅이 적용돼요.


■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■
  문제가 생겼을 때
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

Q) 컬러스페이스/데이터타입 목록이 비어 있어요
A) Nuke에서 setup_pro 패널을 한 번 열어보세요.
   그러면 목록이 채워져요.

Q) OCIO 경로 오류가 나요
A) .ocio 파일이 실제로 그 경로에 있는지 확인하세요.

Q) 프리셋 적용 시 일부 실패 메시지가 떠요
A) Nuke 버전에 따라 다를 수 있어요.
   실패한 항목을 알려주시면 다음 버전에서 수정해 드릴게요.


■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■
  요약 (한 줄로)
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

1. 이 폴더(release) 들어가서 → setup_pro_manager.exe 실행
2. install_to_nuke.bat 실행 → Nuke 재시작
3. 프리셋 저장 → Nuke에서 setup_pro 메뉴로 적용
